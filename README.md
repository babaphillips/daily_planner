## Description:

An application that creates a daily planner where the time blocks will change colors to show if they are in the past, present or future.

## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:babaphillips/daily_planner.git
   ```
2. Run application thru Go Live Extension.

## Usage

This application can be used by anybody who is looking for a daily planner where the time blocks will change if they are in the past, present or future.
